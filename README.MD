# Marvel app 
App that uses Marvel api.

### Tech stack 
- DaggerHilt (2.40.5)
- Retrofit (2.9.0)
- Coroutines (1.5.2)
- Paging (3.1.0)
- Glide (4.12.0)
- Mockk (1.12.2)

### Architecture

App divided in data, domain and features following SOLID principles.
